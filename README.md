## Summary
This repository contains four "Code You Never Write" projects built by
describing real problems clearly to an AI, generating working scripts, and
verifying each result against a fact I already knew to be true. The goal
wasn't to become a coder — it was to become a good client for code.

## AI Tool(s) Used
Claude (Anthropic)

## Projects
These are the following projects this repository contains.
| Folder | Project_Title | Breif |
|---|---|---|
| `project-1-money-detective/` | Money Detective | Finds recurring charges, forgotten subscriptions, and duplicate payments in transaction history |
| `project-2-whats-my-grade/` | What's My Grade, Really | Calculates true current grade using a teacher's exact weighting and drop-lowest rules |
| `project-3-books-dont-match/` | The Books Don't Match | Reconciles a hand-counted total against a messy digital payment record |
| `project-4-organize-the-mess/` | Organize the Mess | Safely finds duplicate/large files and organizes a messy folder, with a mandatory dry-run/approval step before touching anything |

Each folder contains:
- `script.py` — the final working script
- `README.md` — problem, tool used, verification method, findings
- `prompts.md` — the prompts used (initial + follow-ups)
- Output/data files and screenshots showing the working result

## Core Principle Followed Throughout
Never trust output that can't be checked. Every project was verified against
a number, total, or fact known beforehand — if the AI's result didn't match
the known fact, the code was treated as wrong, not the fact.
